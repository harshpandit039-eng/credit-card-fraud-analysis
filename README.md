Project Overview

This project focuses on analyzing credit card transaction data to understand customer spending behavior, detect patterns, and evaluate potential risk indicators.

Dataset Description.

The dataset contains:

Transaction ID

Customer ID

Transaction Date

Transaction Amount

Merchant Category

Payment Method

City / Region

Fraud Status (if available)

 Analysis Performed
1️ Transaction Analysis

Total number of transactions

Monthly transaction trend

Average transaction amount

2️ Customer Spending Behavior

Top spending customers

Spending by region

Spending by merchant category

3️ Fraud / Risk Analysis

Fraudulent transaction percentage

High-risk transaction patterns

Unusual spending behavior detection

4️ Revenue Insights

Total revenue generated

Revenue by category

Peak transaction time
